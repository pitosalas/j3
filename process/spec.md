# Spec for j3

* A package of claude instructions to institute a method for developing programs with claude
* So using j3 as a tool I can more easily create a new app using a good process and methodology
* To keep things clear, we say j3 when we mean this tool and the target app when we mean the app we are building
* To enable a new application with j3 all you have to do is to copy the .j3 directory. Everything needed to enable the methodology is in .j3 and nothing more.
* Produce a clean open-source ready repo for .j3 and also for any target app that it makes
