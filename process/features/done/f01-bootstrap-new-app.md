# Feature description for feature F01
## F01 — Bootstrap a new target app with j3
**Priority**: High
**Done:** yes
**Tests Written:** yes
**Test Passing:** yes
**Description**: When a user copies the `.j3` directory into a new project, Claude should automatically set up the required `process/` folder structure: `spec.md`, `features/notdone/`, `features/done/`, `tasks/notdone/`, `tasks/done/`, and the template files. This makes j3 truly self-contained — copying `.j3` is all that is needed to enable the full methodology on any new app.
