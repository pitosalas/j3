# Feature description for feature F02
## F02 — Produce open-source ready repo structure
**Priority**: High
**Done:** yes
**Tests Written:** yes
**Test Passing:** yes
**Description**: When bootstrapping a new project with j3, the scaffold should produce a clean open-source ready repo. This means including a `LICENSE` file (MIT), a `README.md` template, and a `.gitignore` appropriate for Python projects. The j3 repo itself should also have these files. Everything needed should live in `.j3` so copying it is sufficient.
